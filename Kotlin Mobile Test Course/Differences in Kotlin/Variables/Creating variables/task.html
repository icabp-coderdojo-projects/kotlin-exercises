In languages we have different ways of making new variables. We may be used to Javascript where we can make a variable with: 

  const example = "This is a variable in Javascript"
or with Python, simply: 

  example2 = 23
Any variable we create takes any data we give it.

Kotlin is similar, where we use val and var instead of const and let in Javascript, respectively.

For example: 

  val example = "This is a variable in Kotlin"
  var example2 = 23
Try running the code in the task

In Javascript, when we used const we could not change the value.

 In Kotlin, val acts the same way

When we used let we could change the value whenever we wanted.

 In Kotlin, var acts the same way
